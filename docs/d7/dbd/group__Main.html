void HAL_CAN_RxFifo0MsgPendingCallback ( CAN_HandleTypeDef *  hcan)

Our implementation of the HAL CAN's interrupt callback function, overrides the weak version in stm32f4xx_hal_can.c. This function should be modified if ANY function is designed to be blocked until woken up by a CAN interrupt.

Parameters
hcana pointer to a CAN_HandleTypeDef struct. Probably hcan1 or hcan2.
